# المرجع البصري — شاشة المعاملة المستهدفة

## الملف المطلوب

ضع صورة مرجعية عالية الدقة (مثلاً نموذج **Sales Invoice** الجديد) هنا باسم ثابت:

- **`sales-invoice-target.png`**

يمكن استخدام اللقطة التي تضم:

- شريط علوي (شعار، مسار، بحث)
- عنوان المستند + شارة الحالة + أزرار Save / Submit / Actions
- قسم معلومات الفاتورة على شبكة واضحة (3 أعمدة على سطح المكتب)
- أقسام الدفع والتسليم
- جدول البنود بعرض كامل مع رأس واضح وصفوف متناوبة
- ملخص مالي وملاحظات
- شريط إجراءات سفلي (Print / Email / Save / Submit) إن وُجد في التصميم النهائي

## العلاقة بالتنفيذ في Frappe

- المرجع يوجّه **الهيكل والتسلسل الذهني**؛ التنفيذ الفعلي يمر عبر طبقة التصميم العالمية + تخصيص النماذج عند الحاجة.
- رأس Frappe Desk **لاصق** (`page-head`) — غالباً يغطي جزءاً من مطابقة المرجع دون JS إضافي.
- الشريط السفلي المكرر للإجراءات قد يحتاج مرحلة منفصلة — انظر `../PLAN.md` المرحلة 3.
