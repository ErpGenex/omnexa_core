# رموز التصميم (Design Tokens) — ErpGenex Desk

تُعرّف في `:root` داخل `public/css/omnexa_core.css` بالبادئة `--omnexa-*`.  
الاستخدام: فقط هذه المسافات في CSS الجديد لتبقى الشاشات متسقة.

## المسافات (Spacing)

| الرمز | القيمة | الاستخدام |
|--------|--------|------------|
| `--omnexa-micro` | 4px | تعديلات دقيقة، محاذاة أيقونات صغيرة |
| `--omnexa-space-0` | 8px | تباعد بين تسمية وحقل، كثافة الحقول |
| `--omnexa-space-1` | 8px | (مرادف للسابق للتوافق مع إصدارات سابقة) |
| `--omnexa-space-2` | 12px | حشو بطاقات الأقسام الداخلي |
| `--omnexa-space-3` | 16px | فجوة شبكة بين الأعمدة، هامش قسم |
| `--omnexa-section-gap` | 24px | بين أقسام النموذج الرئيسية |
| `--omnexa-page-gap` | 32px | فصل كتل الصفحة عن الهوامش الخارجية |

## النصف قطر والحدود

| الرمز | القيمة |
|--------|--------|
| `--omnexa-radius` | 8px (بطاقات الأقسام) |
| `--omnexa-radius-input` | 6px (حقول الإدخال ضمن المحسّن) |
| `--omnexa-border` | 1px solid rgba(15, 23, 42, 0.08) |
| `--omnexa-shadow-soft` | ظل خفيف للبطاقات |
| `--omnexa-focus` | حلقة تركيز لوحة المفاتيح |

## الطباعة (Typography) — النطاق المستهدف

| العنصر | الحجم المستهدف | ملاحظة |
|--------|----------------|--------|
| عنوان المستند (page title) | 20–24px bold | ضمن `.omnexa-form-optimized .page-head` |
| عنوان قسم | 16–18px semibold | `.section-head` |
| تسميات حقول | 12–13px | `.control-label` عبر Frappe + وزن 600 في المحسّن |
| إدخال | 13–14px | حقول داخل `.omnexa-form-optimized` |

## الألوان الدلالية (من المحسّن الحالي)

| الاستخدام | ملاحظة |
|-----------|--------|
| حقول مهمة | `.omnexa-important-control` — أزرق داكن للتسمية |
| حقول مطلوبة | `.omnexa-required-control` — علامة حمراء بعد التسمية |

## الشبكة المنطقية

- **سطح المكتب:** `.omnexa-layout-normal` → عمودان؛ `.omnexa-layout-dense` / `.omnexa-layout-complex` → ثلاثة أعمدة كحد أقصى
- **جدول فرعي:** يمتد على كامل عرض القسم (`grid-column: 1 / -1`)
